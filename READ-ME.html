M45 Relay Client Instructions

Getting Started

  1. Launch the program M45-Relay-Client.exe.
    This will connect to the relay server and retrieve a list of available servers.
  2. The client will automatically generate a file called connect-links.html and attempt to open it in your default browser.

If connect-links.html does not open automatically, you can open it manually:

  • Locate connect-links.html in the same folder as this file.
  • Double-click it or open it via this link: connect-links.html

Factorio (Without Steam)

If you do not have Steam installed, you can still connect in Factorio:

  1. In Factorio, go to Multiplayer → Connect to address.
  2. Enter the server address (for example, 127.0.0.1:10000) and click Connect.
